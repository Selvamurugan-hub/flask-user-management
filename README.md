# Flask User Management System

A full-stack web application built using Python Flask and SQLite.

## Features

- User Signup & Login (Authentication)
- Password Hashing (Secure Login)
- Add User
- Edit User
- Delete User
- Search User
- Session Management
- Simple Dashboard UI

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Werkzeug (Password Hashing)

## How to Run

1. Clone the repository
2. Install dependencies:
   pip install flask werkzeug
3. Run the application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000

## Project Type

Mini Full-Stack CRUD Application with Authentication.
